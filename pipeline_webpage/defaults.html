BWA MEM

Flag Default Value Meaning
-t 1 number of reads
-k 19 minimum seed length
-w 100 band width for banded alignment
-d 100 off diagonal X-droppoff
-r 1.5 look for internal seeds inside a seed longer than {-k} * FLOAT
-y 20 seed occurrence for the 3rd round seeding
-c 500 skip seeds with more than INT occurrences
-D 0.50 drop chains shorter than FLOAT fraction of the longest overlapping chain
-W 0 discard a chain if seeded bases shorter than INT
-m 50 perform at most INT rounds of mate rescues for each read
-S NULL skip mate rescue
-P NULL skip pairing; mate rescue performed unless -S also in use
